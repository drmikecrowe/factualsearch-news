  <action>
    Install and configure Tailwind CSS 4 using the @tailwindcss/vite plugin approach:

    1. Install dependencies:
       ```bash
       npm install tailwindcss @tailwindcss/vite
       ```

    2. Update astro.config.mjs to add Tailwind Vite plugin:
       ```javascript
       import { defineConfig } from 'astro/config';
       import tailwindcss from '@tailwindcss/vite';

       export default defineConfig({
         output: 'static',
         site: 'https://factualsearch.news',
         vite: {
           plugins: [tailwindcss()]
         }
       });
       ```

    3. Create src/styles/global.css:
       ```css
       @import "tailwindcss";

       /* Custom theme extensions can go here */
       @theme {
         --font-family-sans: system-ui, -apple-system, sans-serif;
       }
       ```

    4. Create src/layouts directory (will be populated in Plan 01-02)

    DO NOT:
    - Do NOT create tailwind.config.ts or tailwind.config.js
    - Do NOT use PostCSS configuration for Tailwind
    - Do NOT add @astrojs/tailwind (use @tailwindcss/vite instead)
  </action>
