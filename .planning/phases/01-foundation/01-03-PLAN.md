<task type="auto">
  <name>Install and configure @astrojs/sitemap</name>
  <files>package.json, astro.config.mjs</files>
  <action>
    Install and configure the sitemap integration:

    1. Install the integration:
       ```bash
       npm install @astrojs/sitemap
       ```

    2. Update astro.config.mjs to add sitemap integration:
       ```javascript
       import { defineConfig } from 'astro/config';
       import tailwindcss from '@tailwindcss/vite';
       import sitemap from '@astrojs/sitemap';

       export default defineConfig({
         site: 'https://factualsearch.news',
         output: 'static',
         integrations: [
           sitemap()
         ],
         vite: {
           plugins: [tailwindcss()]
         }
       });
       ```

    Key points:
    - `site` property is REQUIRED for sitemap generation (already set in Plan 01-01)
    - sitemap() automatically uses the site URL
    - No additional configuration needed for basic setup

    DO NOT:
    - Do NOT manually create sitemap.xml files
    - Do NOT forget the site property (sitemap silently fails without it)
  </action>
  <verify>
    Commands to run:
    1. `npm run build` - Must succeed
    2. `grep "@astrojs/sitemap" package.json` - Should find dependency
    3. `grep "sitemap()" astro.config.mjs` - Should find integration

    Check astro.config.mjs has:
    - import for @astrojs/sitemap
    - sitemap() in integrations array
  </verify>
  <done>
    - @astrojs/sitemap in package.json dependencies
    - astro.config.mjs imports and uses sitemap()
    - Build succeeds
  </done>
</task>
