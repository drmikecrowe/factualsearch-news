  <action>
Create the MbfcNews.svelte component that fetches and displays MBFC news feed.

Implementation details:
1. Use Svelte 5 with TypeScript (`<script lang="ts">`)
2. Define interfaces for FeedItem and Rss2JsonResponse:
```typescript
interface FeedItem {
  title: string;
  link: string;
  pubDate: string;
  description: string;
  content: string;
  author: string;
  thumbnail: string;
  guid: string;
  categories: string[];
}

interface Rss2JsonResponse {
  status: string;
  feed: { title: string; description: string; };
  items: FeedItem[];
}
```
3. Use Svelte 5 runes:
   - `$state<FeedItem[]>([])` for news array
   - `$state(true)` for loading
   - `$state<string | null>(null)` for error
4. Access API key: `const API_KEY = import.meta.env.PUBLIC_RSS2JSON_API_KEY`
5. RSS URL: `https://mediabiasfactcheck.com/feed/`
6. Use `$effect` with AbortController for cleanup:
   - Encode RSS URL with encodeURIComponent
   - Fetch from RSS feed source
   - Handle response: check status === 'ok', set news array
   - Handle errors: set error message, console.error for debugging
   - Return cleanup function to abort fetch
7. Render:
   - Section wrapper with class "mbfc-news"
   - Title: "Latest Media Bias/Fact Check News" with link to MBFC
   - Loading state, error state, success state
   - List of articles (top 5) with title, date, link
   - Use semantic HTML (article, time with datetime attribute)

DO NOT hardcode the API key - use import.meta.env.PUBLIC_RSS2JSON_API_KEY
DO NOT use React patterns - use Svelte 5 $state and $effect
</action>
